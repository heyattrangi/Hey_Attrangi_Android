# Changelog

All notable changes to the Hey Attrangi Expo app are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2026-07-15 — Release Candidate

### Added
- Full consumer wellness surface: Home, Companion chat, Mood, Therapists, Profile
- AI conversation readiness (streaming UX, templates, memory, escalation placeholders)
- Wellness reports, family/trusted circle, community, therapist/admin portal shells
- Developer tools gallery, feature flags, mock/env switches
- Native splash + adaptive icon configuration; EAS build profiles
- Store metadata drafts and marketing asset folders
- Sprint QA docs (performance, a11y, design compliance, production score)

### Changed
- Brand splash displays **Hey Attrangi** with version marker
- Design-aligned Home Explore (2 cards), Chat “Just listen” shell, Profile design core
- Privacy / Terms help articles structured for RC with hosted URL CTAs

### Known limitations
- Mock services by default (`EXPO_PUBLIC_USE_MOCK`)
- Video session UI remains Design No-Go
- Counsel-approved Privacy Policy / Terms bodies pending
- Store screenshots / feature graphic placeholders only
- EAS `projectId` must be replaced before cloud builds
