# Hey Attrangi

Expo frontend application for Hey Attrangi. Mock-first; no backend in this repository.

## Quick start

```bash
cd expo-app
npm install
npx expo start
```

Scan the QR code with Expo Go.

## Demo tips

- Sign in with any email/phone + password
- OTP: enter any 6-digit code
- Google Sign-In uses a mock token (no native Google SDK in Expo Go)

See [`expo-app/README.md`](expo-app/README.md) for more details.
